# Pagina Presentacion

Portafolio de presentación realizada con html y css.

## Requisitos

- [Git](https://git-scm.com/)
- [Visual Studio Code](https://code.visualstudio.com/) (o cualquier otro editor de texto)
- [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extensión para Visual Studio Code (opcional)

## Instrucciones de Uso

1. **Clona el repositorio:**

   ```bash
   git clone https://github.com/SrPabvliss/PaaginPresentacion.git
   ```

2. **Abre el proyecto en tu editor de texto:**

   ```bash
   cd PaaginPresentacion
   code .
   ```

3. **Instala la extensión Live Server (si estás utilizando Visual Studio Code):**

   - Instala la extensión desde [Visual Studio Code Marketplace](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer).

4. **Abre el archivo HTML principal (`index.html`) con Live Server:**

   - Haz clic derecho en el archivo y selecciona "Open with Live Server".

5. **Visualiza el proyecto:**

   - El proyecto se abrirá automáticamente en tu navegador predeterminado.

## Estructura del Proyecto

- `index.html`: Archivo principal HTML.
- `css/`: Carpeta que contiene los archivos CSS.
- `img/`: Carpeta que contiene las imágenes utilizadas.
